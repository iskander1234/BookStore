Дата возврата не может быть продлена

Дата возврата книги, которую вы пытаетесь продлить, не может быть продлена. Возможные причины:

- Достигнут максимальный срок использования.
- Срок возврата уже продлен на 3 недели.
- Кто-то зарезервировал книгу.

Не забудьте вовремя вернуть эту книгу в библиотеку.
